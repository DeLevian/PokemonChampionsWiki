# Personalizzazione Grafica Pokémon Champions

Questa cartella contiene i file necessari per personalizzare l'aspetto del portale in futuro.

## Contenuti
- `bg.png`: Immagine di sfondo dello stadio. Puoi sostituire questo file con un'altra immagine mantenendo lo stesso nome per cambiare lo sfondo del sito.
- `custom_vars.css`: Contiene le variabili CSS principali (colori, font, bordi). Modifica i valori in questo file per cambiare l'identità visiva globale senza toccare il codice principale.

## Come Sostituire lo Sfondo
1. Carica una nuova immagine in questa cartella.
2. Nominala `bg.png` (sovrascrivendo quella esistente) oppure aggiorna il percorso nel file `src/styles/main.css`.

## Sviluppi Futuri
In questa cartella potrai aggiungere:
- `logo.png`: Per il brand del sito.
- `button_styles.css`: Per stili personalizzati dei pulsanti.
- Icone custom per i tipi o le categorie.
